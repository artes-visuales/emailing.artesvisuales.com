Curso Diseño Web con WordPress
El Curso WordPress Esencial, está pensado en aquel usuario que desee adentrarse en el mundo del desarrollo de sitios web y el comercio electrónico mediante el CMS más utilizado del mundo.

Objetivos: El objetivo fundamental del curso es aprender a manejar las herramientas esenciales del CMS para la creación de websites dinámicas. Del mismo modo, adquiriremos conocimientos sólidos acerca de la plataforma de desarrollo WordPress, tales como su instalación, configuraciones generales, creación de contenidos, publicaciones… y trataremos temas relativos a la implementación de funcionalidades extra con el uso de Plugins y Temas en los sitios que creemos. Método de aprendizaje: Integramente online mediante campus virtual, soporte en 24 h, hosting web para pruebas, libros, documentos pdf, explicaciones audiovisuales, test, ejercicios prácticos y proyectos de asignatura y final del curso.

Módulos del curso: WordPress Esencial, Wordpress Seguridad, WordPress Diseño Themes, Wordpress ecommerce.

Estudio práctico
WordPress Esencial
Adquiriremos los conocimientos básicos acerca de la plataforma de desarrollo WordPress, tales como su instalación, configuraciones generales, creación de contenidos, publicaciones…
Estudio práctico
WordPress Themes
Aprenderemos a crear un Sitio web desde cero, con un diseño, estructura y contenidos completamente personalizados, que nos servirán de guía a la hora de desarrollar plantillas propias en WordPress.
Estudio práctico
WordPress Seguridad
El objetivo principal del curso, es establecer buenas prácticas y comportamientos, que permitan una alta perdurabilidad de nuestro sitio web frente a los ataques de los hackers.
Estudio práctico
WordPress Ecomerce
Gracias a este curso aprenderemos el modo de convertir un sitio web realizado con la plataforma de publicación WordPress, en una tienda de comercio electrónico.
